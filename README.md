# Jogo Do Número Secreto

## Descrição

Projeto desenvolvido como parte do curso de Lógica de Programação em JavaScript da Alura. O jogo desafia o usuário a adivinhar um número secreto através de dicas de "maior" ou "menor".

## Recursos Principais

- **API ResponsiveVoice**: Integração de síntese de voz para feedback auditivo, tornando a experiência mais interativa e acessível
- Lógica de jogo implementada em JavaScript puro
- Interface responsiva com validação de entrada

## Estrutura do Projeto

Todo o desenvolvimento da lógica do jogo foi realizado no arquivo `app.js`, incluindo:

- Geração do número secreto
- Validação de tentativas
- Feedback visual e auditivo ao usuário
- Gerenciamento do estado do jogo

## Créditos

- **Artes e design inicial**: Material do curso de Lógica de Programação em JavaScript - Alura
- **Desenvolvimento**: Implementação da lógica de jogo e integração da API ResponsiveVoice

## Como Usar

1. Abra o aplicativo no navegador
2. Insira seu palpite e clique em "Chutar"
3. Siga as dicas de voz para encontrar o número secreto
